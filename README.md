# Orca Sensor Marketplace

Claude Code plugin marketplace.

## Plugins

### claude-toolkit

MCP tool routing, native tool enforcement, edit guards, and session analytics.

```bash
claude plugin marketplace add orca-sensor-marketplace ilyabrykau-orca/orca-sensor-marketplace
claude plugin install claude-toolkit@orca-sensor-marketplace
```

See [claude-toolkit](https://github.com/ilyabrykau-orca/claude-toolkit) for details.
